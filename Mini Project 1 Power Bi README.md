# 📊 Supermart Grocery Sales Retail Analysis


## 📌 Project Overview

This Power BI Mini Project focuses on analyzing Supermart Grocery Sales data to gain valuable business insights. The dashboard helps track sales performance, profit trends, customer orders, and regional performance through interactive visualizations.

---

## 🛠️ Tools Used

- Microsoft Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization

---

## 📂 Dataset Details

The dataset contains the following information:

- Order ID
- Order Date
- Region
- City
- Category
- Sub Category
- Sales
- Profit
- Discount
- Quantity

---

## 📈 Key Performance Indicators (KPIs)

| KPI | Value |
|------|------|
| Total Profit | 3.43M |
| Total Orders | 9153 |
| Total Sales | 13.69M |
| Total Discount | 2.08K |
| Profit Margin | 25% |

---

## 📊 Dashboard Visualizations

### 1. Total Sales by Region
Displays sales performance across different regions.

### 2. Total Profit by Sub Category
Shows which product subcategories generate the highest profit.

### 3. Total Sales by Month
Analyzes monthly sales trends throughout the year.

### 4. Total Sales by Category
Displays category-wise sales distribution.

### 5. Total Sales by Sub Category
Compares sales among different product subcategories.

### 6. Total Orders by City
Shows city-wise order distribution.

---

## 🎛️ Filters Used

- Region Filter
- Category Filter
- Order Date Filter

These filters allow users to interactively explore and analyze the data.

---

## 🔍 Key Insights

- West region achieved the highest sales.
- Soft Drinks generated the highest sales among subcategories.
- Sales showed significant growth during October, November, and December.
- The business maintained a profit margin of 25%.
- Major cities generated more than 400 orders.

---

## ✅ Conclusion

The Supermart Grocery Sales Retail Analysis Dashboard provides valuable insights into sales, profit, and customer purchasing behavior. It helps businesses monitor performance, identify trends, and make data-driven decisions.
