## Project Overview

This Power BI project analyzes grocery retail sales performance across regions, categories, and cities.

## Tools Used

* Power BI Desktop
* Power Query
* DAX

## Key Metrics

* Total Sales: 13.69M
* Total Profit: 3.43M
* Total Orders: 9153
* Profit Margin: 25%

## Charts Used

* Sales by Region
* Profit by Subcategory
* Sales by Month
* Sales by Category
* Orders by City

## Key Insights

* West region has the highest sales.
* Soft Drinks generate the highest profit.
* Sales peak during October to December.
* Kanyakumari has the highest number of orders.

## Conclusion

The dashboard provides valuable insights into sales performance and helps support data-driven business decisions.
