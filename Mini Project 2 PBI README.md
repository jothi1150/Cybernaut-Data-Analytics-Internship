# 🏥 Health and Medical Analysis Dashboard

## 📌 Project Overview
The Health and Medical Analysis Dashboard is an interactive Power BI dashboard developed to analyze healthcare data and generate meaningful insights related to patients, hospitals, doctors, insurance providers, and medical conditions.

This dashboard enables healthcare organizations to monitor performance, improve patient care, and make data-driven decisions through interactive visualizations.

---

## 🎯 Project Objectives
- Analyze healthcare and patient-related data.
- Monitor key healthcare performance indicators.
- Evaluate revenue generated by insurance providers.
- Track patient test results and medical conditions.
- Improve healthcare decision-making through data visualization.

---

## 📊 Dashboard Highlights

### 🔹 Key Metrics
- 💰 Total Revenue: **1.42 Billion**
- 👨‍⚕️ Total Patients: **56K**
- 🩺 Total Doctors: **40K**
- 🏥 Average Stay Duration: **15.51 Days**
- 🛏️ Total Bed Space: **400**

### 🔹 Interactive Filters
- Admission Type
- Age
- Blood Type
- Gender
- Medication
- Room Number

### 🔹 Visualizations Included
- Patient Test Results Analysis
- Average Billing Trend by Year
- Revenue by Insurance Provider
- Patients and Average Billing by Medical Condition
- Doctor Information Analysis
- Hospital Information Analysis

---

## 🛠️ Technologies Used
- Microsoft Power BI
- Data Cleaning & Transformation
- Data Modeling
- DAX (Data Analysis Expressions)
- Interactive Dashboard Design

---

## 📈 Key Insights
- Analyzed patient distribution based on test results.
- Compared insurance providers by revenue contribution.
- Identified trends in average billing across years.
- Evaluated patient and billing patterns by medical conditions.
- Monitored healthcare resource utilization effectively.
