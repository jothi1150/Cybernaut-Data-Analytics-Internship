# 📊 Social Media Analytics Dashboard - Power BI

## 🚀 Project Overview

The **Social Media Analytics Dashboard** is a Power BI project designed to analyze and visualize social media performance across multiple platforms. This dashboard helps businesses, marketers, and content creators monitor engagement metrics and make data-driven decisions.

The project transforms raw social media data into meaningful insights by analyzing likes, comments, shares, and overall engagement across different social media platforms.

---

## 🎯 Objectives

* Analyze social media engagement metrics.
* Compare platform performance.
* Identify top-performing content.
* Track audience interaction trends.
* Support data-driven marketing decisions.

---

## 📂 Dataset Information

The dataset contains social media performance data with the following fields:

* Platform Name
* Likes
* Comments
* Shares
* Post Date
* Content Category

**Dataset Size:** Approximately 10,000 records.

---

## 🛠 Tools & Technologies Used

* **Power BI**
* **Power Query**
* **Microsoft Excel / CSV**
* **Data Cleaning & Transformation**
* **Data Visualization**

---

## 📈 Key Metrics

The dashboard calculates and visualizes:

* 👍 Total Likes
* 💬 Total Comments
* 🔄 Total Shares
* 📊 Total Engagement
* 📉 Average Engagement

---

## 📊 Dashboard Features

### KPI Cards

Displays total engagement statistics and key performance indicators.

### Platform Analysis

Compare engagement across:

* Instagram
* Facebook
* Twitter
* YouTube

### Category Analysis

Analyze engagement by content categories such as:

* Entertainment
* Lifestyle
* Education
* Technology
* News

### Top Performing Posts

Identify the best-performing posts based on engagement metrics.

### Interactive Filters

Filter data by:

* Platform
* Date
* Category

## 📌 Key Insights

* Instagram showed the highest engagement among platforms.
* Video content generated significantly higher engagement than static content.
* Engagement varied across different months and content categories.
* Certain platforms consistently outperformed others in audience interaction.
